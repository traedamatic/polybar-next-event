# ploybar next event

This polybar fetches the a calender and shows in the polybar the next calender. It adds colors to it 1 hour away, green, half hour yellow, 15minutens red. All color codes are light nice color. On click on the event a dialog open on the left side details for the events on right side the upcomping events

## Features

1. Show next event in the ploybar
2. Show Dialog on click with details and next events
3. Simple color for easy overview
